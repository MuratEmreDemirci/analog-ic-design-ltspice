# analog-ic-design-ltspice
Analog IC design projects implemented in LTspice, including OTA and two-stage unbuffered op-amp. Features complete analysis: DC operating point, gain-bandwidth, phase margin, slew rate, ICMR, and output swing.
# 🔬 Analog IC Design (LTspice)

This repository contains analog integrated circuit design projects implemented in LTspice.

---

## 📁 Projects

### 🔹 OTA (Operational Transconductance Amplifier)

* MOSFET differential pair
* Current mirror load
* High gain analog design

### 🔹 Two-Stage Unbuffered Op-Amp

* Miller compensation
* Phase margin optimization
* High gain and bandwidth design

---

## ⚙️ Analyses Performed

* DC Operating Point (.op)
* AC Analysis (Bode Plot)
* Transient Analysis (Slew Rate)
* ICMR (Input Common-Mode Range)
* Output Swing

---

## 📊 Key Results

### OTA

* High gain (gm·ro based)
* Slew rate limited behavior observed
* MHz-level bandwidth

### Two-Stage Op-Amp

* Gain ≈ 7000 V/V (≈76 dB)
* Phase Margin ≈ 60°
* Slew Rate ≈ 7.75 V/µs
* Power Dissipation ≈ 0.94 mW
